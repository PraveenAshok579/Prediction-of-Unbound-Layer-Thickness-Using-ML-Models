Unbound Layer Thickness Prediction Web Application

This Streamlit app demonstrates ensemble machine learning models
for predicting unbound layer thickness.

Models included:
- Random Forest
- Extra Trees
- Gradient Boosting
- XGBoost
- AdaBoost

Evaluation Metrics:
- R²
- RMSE
- MAPE
- 5-Fold Cross Validation R²

Deployment:
1. Install dependencies:
   pip install -r requirements.txt

2. Run locally:
   streamlit run app.py
